A plugin for sending sms notification after placing orders in WooCommerce using TalkWithText SMS gateway

== Description ==

This is a WooCommerce add-on. Using this plugin user can get SMS alert when place an order in WooCommerce site

= Key Features =

* Integrated TalkwithText SMS Gateways
* Admin can get SMS notifications when place an order
* Buyer can get SMS notifications when place an order
* Available settings for Admin to control SMS settings and gateways
* Customizable SMS text for different order status
* Directly contact with buyer via SMS.
* Admin can force to buyer to get SMS notifications.
* Send Order details ( order no, order status, order amount and order items ) in SMS text
* Extended Settings Option

== Installation ==

= New Install via FTP =

1. Unzip the package talkwithtext-sms-notification-for-woo.zip on you computer.
2. Upload the unzipped folder the /wp-content/plugins/ directory.
3. Activate the plugin through the Plugins menu in WordPress.
4. Goto SMS Settings menu and set your options.

= New Install via the WordPress uploader =

1. Click Plugins > Add New inside of your WordPress install.
2. Click Upload and select the package talkwithtext-sms-notification-for-woo.zip
3. Activate the plugin through the Plugins menu in WordPress.
4. Goto SMS Settings menu and set your options.

= Updating via the FTP =

1. If you've made any custom changes to the core you'll need to merge those changes into the new package.
2. Unzip the package talkwithtext-sms-notification-for-woo.zip on you computer.
3. Replace the current folder in your plugins folder with the new unzipped folder from your computer.
